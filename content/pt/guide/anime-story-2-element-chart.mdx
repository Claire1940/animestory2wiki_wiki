<Section icon={Map} title="Fluxo de Trabalho Ideal para Montagem de Equipe">

Montar uma equipe em Anime Story 2 exige equilibrar sua cobertura elemental com bônus de Líder. Se sua equipe for inteiramente de um único elemento, você será bloqueado por inimigos que o contra-atacam na **tabela de elementos de anime story 2**.

<Steps>
  <Step num="1" title="Selecione uma Unidade Líder">
    Escolha uma unidade com uma Habilidade de Líder poderosa. Por exemplo, use **Mazara** se você tiver unidades Ninja ou **Dior** se estiver usando uma equipe de farm temática de Maids.
  </Step>
  <Step num="2" title="Verifique a Cobertura Elemental">
    Certifique-se de ter pelo menos dois elementos diferentes em seus slots de DPS. Isso evita que você fique travado contra um chefe que resiste ao seu tipo de dano principal.
  </Step>
  <Step num="3" title="Equipe Relíquias e Atributos (Traits)">
    Para 2026, a **Relíquia Taco de Beisebol (Baseball Bat)** é o padrão ouro para DPS. Para atributos, **Monarch** é a melhor escolha geral, enquanto **Solar** é preferido para Tanques.
  </Step>
  <Step num="4" title="Otimize para o Modo de Jogo">
    Troque suas unidades com base na atividade. Use **Greatest Maid** para farm de Portais para obter o bônus de 10% na taxa de drop, mas mude para **Black Bull** para subir andares altos na Tower.
  </Step>
</Steps>

<Callout type="info" title="Dica de Relíquia">
Use as relíquias **Demonring** ou **Coal** em seus Tanques para aumentar a sobrevivência deles durante as raids de Chefes Mundiais.
</Callout>

</Section>
